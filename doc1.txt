班级留言板云平台综合实训部署文档 
本文档用于“云平台综合实训”课程。目标是让学生通过一个 Web 项目，完成 Docker 容器化、Docker Compose 编
排、Kubernetes 部署、服务访问、配置修改、滚动更新和基础排错。
1. 项目目标
 
本项目名为“班级留言板”。最终效果如下：
打开网页后可以看到留言输入框。
输入姓名和留言内容后点击提交。
页面可以显示已经提交的留言。
后端提供健康检查接口和留言接口。
项目可以用 Docker 单独运行，也可以用 Docker Compose 一键运行。
项目可以部署到 Kubernetes 中，通过 Service 对外访问。
2. 准备环境
 
请提前安装以下软件：
Docker Desktop
Node.js，建议 18 或以上
kubectl
Kubernetes 环境，可以使用 Docker Desktop 自带 Kubernetes、Minikube 或学校提供的集群
验证命令：
如果使用 Docker Desktop Kubernetes，请在 Docker Desktop 设置中启用 Kubernetes。

const app = express();
const port = process.env.PORT || 3000;
app.use(cors());
app.use(express.json());
let messages = [
  {
    name: "老师",
    content: "欢迎来到云平台综合实训。",
    time: new Date().toLocaleString("zh-CN")
  }
];
app.get("/health", (req, res) => {
  res.json({ status: "ok", service: "message-board-api" });
});
app.get("/messages", (req, res) => {
  res.json(messages);
});
app.post("/messages", (req, res) => {
  const name = String(req.body.name || "").trim();
  const content = String(req.body.content || "").trim();
  if (!name || !content) {
    return res.status(400).json({ message: "姓名和留言内容不能为空" });
  }
  const message = {
    name,
    content,
    time: new Date().toLocaleString("zh-CN")
  };
  messages.unshift(message);
  res.status(201).json(message);
});
app.listen(port, () => {
  console.log(`message-board-api running on port ${port}`);
});

创建 config.js：
const config = window.APP_CONFIG || {};
const apiBaseUrl = config.API_BASE_URL || "http://localhost:3000";
document.getElementById("title").textContent = config.APP_TITLE || "班级留言板";
document.getElementById("welcome").textContent =
  config.WELCOME_TEXT || "请提交你的实训留言。";
const nameInput = document.getElementById("name");
const contentInput = document.getElementById("content");
const submitButton = document.getElementById("submit");
const messagesSection = document.getElementById("messages");
async function loadMessages() {
  const response = await fetch(`${apiBaseUrl}/messages`);
  const messages = await response.json();
  messagesSection.innerHTML = messages
    .map(
      (item) => `
        <article class="message">
          <strong>${item.name}</strong>
          <p>${item.content}</p>
          <div class="meta">${item.time}</div>
        </article>
      `
    )
    .join("");
}
submitButton.addEventListener("click", async () => {
  const name = nameInput.value.trim();
  const content = contentInput.value.trim();
  if (!name || !content) {
    alert("请填写姓名和留言内容");
    return;
  }
  await fetch(`${apiBaseUrl}/messages`, {
    method: "POST",
    headers: {
      "Content-Type": "application/json"
    },
    body: JSON.stringify({ name, content })
  });
  contentInput.value = "";
  await loadMessages();
});
loadMessages().catch(() => {
  messagesSection.innerHTML = "<p>留言加载失败，请检查后端服务是否启动。</p>";
});

页面打不开：检查 message-frontend 的 Pod 是否 Running，检查访问端口是不是 30081。
留言加载失败：检查 message-api 的 Pod 是否 Running，检查 http://localhost:30080/health 是否可访问。
Pod 一直 ImagePullBackOff：检查镜像名是否正确，检查镜像是否已经构建。
修改配置后页面没变化：重启前端 Pod 或执行滚动重启。
